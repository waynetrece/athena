# Athena 韓境美學 — 提案網站設計規格

## 概述

為韓國醫美代辦品牌「Athena 韓境美學」建立提案展示網站，包含簡報、首頁 Demo、預約系統 Demo。用於 2026-04-16 客戶拜訪展示。

## 技術棧

- Next.js 15 (App Router) + TypeScript + Tailwind CSS 4
- Motion (Framer Motion) — 元件動畫、hover、轉場
- GSAP + ScrollTrigger — 捲動動畫、時間軸、SplitText
- 部署：Vercel

## 設計風格

S22 奢華黑金

```
背景：#000000 / #0A0A0A
金色：#D4AF37（主色）/ #C5A037（輔助）
文字：金色標題 + #E5E5E5 內文
圓角：0px（sharp edges）
邊框：0.5px 金色細線
陰影：gold glow — 0 0 20px rgba(212,175,55,0.3)
字體：serif 標題（Playfair Display）+ sans 內文（Inter）
動效：極慢、從容的淡入
```

## Part 2: 首頁 Demo `/`

每區回答訪客一個問題，解決現站「看不出在做什麼」的核心問題。

### Section 1: Hero
- 訪客問題：「這是什麼？跟我有關嗎？」
- 全屏深色背景 + 微粒金箔 texture
- 主標：「韓國頂尖醫美 台灣安心出發」（GSAP SplitText 逐字淡入）
- 副標：「Athena 韓境美學 — 您通往韓國頂尖醫美的安心橋樑」
- 雙 CTA：[立即預約諮詢] [了解服務流程]
- 動效：文字逐字顯現、按鈕金色光暈

### Section 2: 痛點共鳴
- 訪客問題：「我想去韓國做，但...」
- 3 張卡片：語言不通怎麼辦 / 不知道找哪間診所 / 術後出問題誰負責
- 動效：GSAP ScrollTrigger 觸發、Motion stagger 滑入

### Section 3: 為什麼選 Athena（4 個優勢）
- 訪客問題：「你們跟別人差在哪？」
- KFDA 認證合作 / 全程專人陪同 / 中韓醫療翻譯 / 雙重保險保障
- 動效：數字計數動畫（Motion useMotionValue）

### Section 4: 服務流程時間軸
- 訪客問題：「具體怎麼運作？」
- 視覺化 Day 1-5 時間軸
- Day 1 諮詢評估 → Day 2 出發入住 → Day 3-4 手術休養 → Day 5 觀光回程
- 動效：GSAP ScrollTrigger pin + 逐步亮起（灰→金）+ SVG 連接線描繪

### Section 5: 療程項目卡片
- 訪客問題：「你們能做什麼？」
- 3-4 張卡片：眼部整形 / 鼻部整形 / 微整注射 / 私密整形
- 動效：hover scale + gold glow shadow

### Section 6: 醫師團隊
- 訪客問題：「誰來操刀？靠譜嗎？」
- PASCAL Clinic 醫師卡片（頭像 placeholder + 名字 + 專長）
- 動效：stagger 入場、hover 展開資歷

### Section 7: FAQ
- 訪客問題：「費用？恢復期？安全嗎？」
- 5 題手風琴：費用範圍 / 手術安全 / 語言溝通 / 恢復時間 / 住宿安排
- 動效：AnimatePresence 展開收合

### Section 8: CTA
- 訪客問題：「我要怎麼開始？」
- 「準備好開始您的蛻變之旅了嗎？」
- [預約免費諮詢] 按鈕 → 連到 /booking
- 動效：金色漸層脈動背景

### 全域元素
- 頂部 Navbar：Logo + 錨點導航 + [預約諮詢] CTA
- 捲動進度條（頂部金色細線）
- LINE 浮動按鈕（右下角）
- Footer：聯絡資訊 + 社群 + 地址

## Part 3: 預約 Demo `/booking`

3 步驟表單（簡化現有超長諮詢表）：

### Step 1: 選服務
6 張圖卡：眼部整形 / 鼻部整形 / 臉部微整 / 體雕抽脂 / 私密整形 / 皮膚美容

### Step 2: 選諮詢方式 + 時間
- 3 種方式：線上視訊 / LINE 通話 / 到店面談
- 日曆選日期 + 時段選擇

### Step 3: 基本資料
- 姓名、電話、LINE ID（選填）、備註（選填）
- 極簡 4 欄位，降低門檻

### 完成頁
- 預約確認資訊 + [加入 LINE] + [返回首頁]

### 會員中心概念（/booking 頁底部展示）
- 行程進度時間軸（8 步驟從諮詢到回程）
- 文件管理區 + 顧問聯絡區
- 靜態 mock，展示概念
